¡Bienvenido a Sicefa Sucursal!



Estamos encantados de tenerte aquí. Este sistema integral te brinda acceso a cinco módulos esenciales para la gestión eficiente de tu farmacia:

Empleados: Administra la información de tu equipo, sus roles y responsabilidades.
Clientes: Gestiona la base de datos de clientes para ofrecer un servicio personalizado.
Productos: Mantén un control completo sobre tu inventario y la información de los productos disponibles.
Sucursales: Si tienes múltiples ubicaciones, gestiona fácilmente la información de cada sucursal.
Ventas: Registra y supervisa las transacciones para un seguimiento preciso de las operaciones diarias.

Para acceder a cualquiera de los cinco modulos dirigete al menu lateral izquiero y posiciona tu mouse sobre el modulo al cual quieres acceder y presiona con clic izquiero.
